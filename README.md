# Source-Han-Sans-Monaco
## 思源黑体+Monaco的混合字体

### 来源
看到Ace Editor将中文渲染为2个英文字母大小，严格对齐，实在很爽，就有了这个字体

提供 ***未调整版*** 和 ***调整版***


### 更新
`adjusted-rm-lgt-f`分支去除了字母f相关的**ligature**(连字特性)，否则在Editor/IDE/其他中即使关闭了ligature，也会显示为连字，严重影响观感
